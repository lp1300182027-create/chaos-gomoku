# 离谱成语技能五子棋

本地网页小游戏，不依赖外部服务。

## 运行

直接用浏览器打开 `index.html` 即可。也可以在本目录启动本地服务器：

```bash
python3 -m http.server 5173
```

然后访问 `http://localhost:5173`。

## 玩法

- 15x15 棋盘，五子连线获胜。
- 支持双人互殴和人机练手。
- 每回合获得棋气，棋气可释放离谱四字成语技能。
- 技能包括：飞沙走石、静如止水、力拔山兮、两极反转、偷梁换柱、指鹿为马、无中生有、借尸还魂、围魏救赵、画地为牢、反客为主、天降正义。
- 强控和直接改棋类技能消耗更高，避免技能轰炸盖过五子棋本体。
